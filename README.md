# Bank Management System (Core Java)

## Description

Bank Management System is a console-based application developed using Core Java. The project demonstrates Object-Oriented Programming principles and provides basic banking functionalities through a menu-driven interface.

## Features

* Create Savings and Current Accounts
* Search Account by Account Number
* Deposit Money
* Withdraw Money
* Transfer Money Between Accounts
* Delete Account
* Display All Accounts
* Prevent Duplicate Account Creation

## Technologies Used

* Java
* Collections Framework (ArrayList)
* Object-Oriented Programming
* Eclipse IDE


## How to Run

1. Clone the repository.
2. Open the project in Eclipse or any Java IDE.
3. Compile the project.
4. Run `MainPage.java`.
5. Select the required option from the menu.
