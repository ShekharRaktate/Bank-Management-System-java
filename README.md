# Bank Management System (Console Application)

## Description
A console-based Bank Management System built using Core Java to practice Object-Oriented Programming concepts.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Search Account
- Display All Accounts

## Concepts Used
- Classes and Objects
- Constructors
- Encapsulation
- Inheritance
- Polymorphism (basic structure)
- ArrayList
- Exception Handling
- Menu-driven application

## Technologies
- Java
- Eclipse IDE

## Future Improvements
- Transfer Money
- Delete Account
- File Handling
- JDBC with MySQL
- Savings & Current Account implementation
